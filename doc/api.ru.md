# API ключ для BSN Expert

Руководство для пользователя: как получить персональный API ключ, чтобы синхронизировать контакты с другими сервисами.

## Где получить
- Зайдите на https://bsn.expert и авторизуйтесь.
- Откройте «Настройки» → «API ключи» (https://bsn.expert/preferences/api).
- Нажмите «Создать ключ»:
  - Название — просто подсказка, где вы собираетесь использовать ключ.
  - Разрешения по умолчанию: Читать/Создавать/Изменять включены, «Удалять» выключено. При необходимости скорректируйте галочки.
- После создания ключ показывается целиком **только пока им ни разу не пользовались**. Скопируйте и сохраните.

## Просмотр и управление
- Список ключей виден на той же странице.
- Пока ключ ни разу не использовали, он отображается целиком. После первого использования показывается только маска (первые/последние символы).
- Кнопка «Удалить» полностью отзывает ключ; при утечке сразу удалите и создайте новый.

## Авторизация в API
- Используется заголовок `Authorization: Bearer <ваш_ключ>`.

## Тестовый вызов
- `GET https://bsn.expert/preferences/api` — проверяет ключ и отдаёт JSON со статусом и деталями ключа.

Дальнейшая работа с API (синхронизация контактов) описана в `contacts_sync.ru.md`.
